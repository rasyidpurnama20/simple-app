# Tutorial Aktor

Tutorial ini mengikuti empat peran utama aplikasi. Semua langkah menggunakan fitur yang tersedia pada baseline saat ini: autentikasi, dashboard capaian, katalog kurikulum, tugas, kemajuan pribadi, dan Semantic Analytics API.

| Aktor | Fokus utama | Tutorial |
|---|---|---|
| Prodi | kurikulum, gate aktivasi, approval | [Tutorial Prodi](prodi.md) |
| GPM | verifikasi mutu dan capaian | [Tutorial GPM](gpm.md) |
| Pengampu | pemetaan CPMK dan bukti asesmen | [Tutorial Pengampu](pengampu.md) |
| Mahasiswa | tugas dan kemajuan akademik pribadi | [Tutorial Mahasiswa](mahasiswa.md) |

Fungsi DPA, koordinator, pembimbing, penguji, mentor, dan TPMF merupakan assignment terbatas, bukan akun utama. Assignment tersebut mengikuti scope objek, periode, aksi, dan expiry dari akun utama yang ditugaskan.
